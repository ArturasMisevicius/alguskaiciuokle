# Requirements

- PHP >= 8.2
- Composer >= 2.5
- Node.js >= 18 and npm >= 9 (if building frontend assets)
- Database: MySQL 8 / MariaDB 10.6+ / PostgreSQL 13+ / SQLite (dev)
- Extensions: OpenSSL, PDO, Mbstring, Tokenizer, XML, Ctype, JSON, BCMath, Fileinfo
- Tools: Git, Make (optional on Windows via GnuWin/MSYS), Redis (optional)

Recommended:
- Docker Desktop (optional)
- Mailhog/Postal/SMTP server for email testing
