# Requirements

- PHP >= 8.2 (Laravel 11+)
- Composer >= 2.5
- Node.js >= 18 and npm >= 9 (Vite/Tailwind build)
- Database: SQLite (dev)
- PHP extensions: OpenSSL, PDO, Mbstring, Tokenizer, XML, Ctype, JSON, BCMath, Fileinfo
- Tools: Git; Redis (queues, optional)
- Storage: local
- Auth: Laravel Fortify (password) + optional magic-link
- I18n: English, Lithuanian

Recommended
- Mail testing: Mailhog/Postal/SMTP server
- Stripe account (billing)
