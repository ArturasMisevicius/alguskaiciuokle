# Environment

Key `.env` variables:

- APP_NAME, APP_ENV, APP_KEY, APP_URL
- LOG_CHANNEL, LOG_LEVEL
- DB_CONNECTION, DB_HOST, DB_PORT, DB_DATABASE, DB_USERNAME, DB_PASSWORD
- CACHE_DRIVER, QUEUE_CONNECTION, SESSION_DRIVER
- MAIL_MAILER, MAIL_HOST, MAIL_PORT, MAIL_USERNAME, MAIL_PASSWORD, MAIL_ENCRYPTION, MAIL_FROM_ADDRESS, MAIL_FROM_NAME
- REDIS_HOST, REDIS_PORT, REDIS_PASSWORD (if used)

Profiles:
- Local: `APP_ENV=local`, `APP_DEBUG=true`
- Staging/Prod: `APP_DEBUG=false`, set proper `APP_URL`

Never commit real secrets. Use `.env` or secrets manager.
